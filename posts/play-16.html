Por Dentro do Play 16 — Playwright API Test

Neste Play, exploramos o uso do Playwright para testar APIs RESTful diretamente, sem depender de interfaces gráficas.

Objetivos

  • Realizar requisições HTTP (GET, POST, PUT, DELETE) automatizadas.
  • Validar respostas de status HTTP e payloads JSON.
  • Simular cenários reais de APIs (criação, atualização e remoção de dados).

Ferramentas e Tecnologias

  • Playwright (Node.js framework)
  • REST APIs públicas ou locais
  • Node.js ambiente configurado

Resultados Esperados

  • Requisições automatizadas 100% validadas com assertivas de status e dados.
  • Log de sucesso ou falha no terminal após execução dos testes.
  • Simulação de fluxo CRUD completo em API.
