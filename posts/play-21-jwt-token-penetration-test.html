Play 21 Jwt Token Penetration Test

Este Play executa o comando set -euo pipefail, permitindo avaliar aspectos específicos de segurança.

Script run.sh

#!/usr/bin/env bash
set -euo pipefail

TOKEN="${1:-}"
URL="${2:-http://localhost/api}"
echo "[INFO] Iniciando Play 21 — JWT Token Penetration Test"
python3 jwt_tester.py --token "$TOKEN" --url "$URL"
echo "[SUCESSO] Teste de penetração JWT concluído."
