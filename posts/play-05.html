Por Dentro do Play 05 — QA Automação

Neste Play, utilizamos o Cypress.io para automação de testes de interface (UI) em aplicações web, garantindo funcionalidades críticas.

Objetivos

  • Automatizar fluxos de testes de usuários finais.
  • Validar funcionamento correto de formulários, navegação e interações.
  • Detectar regressões automaticamente.

Comandos Utilizados

  • Rodar testes no navegador: npx cypress open
  • Rodar testes headless (CI/CD): npx cypress run

Resultados Esperados

  • Relatórios de execução dos testes.
  • Prints de falhas e vídeos gravados automaticamente.
  • Confirmação da estabilidade do sistema após mudanças.
